Welcome to Turtle Linux

Turtle linux is a linux distribution focused in python development and python software. It comes python games, found in /usr/games/, code editors (editra and medit), quodlibet as the default music player and mplayer2. There is also, bundled, a gaming interface called vapor. Vapors idea is similar to that of Steam. It is just a interface where you can browse your python games and run them with a click. Just type run_vapor.py to bring it up!

You might notice there is no Office package pre-installed. Well, if you feel you need one, just apt-get install libreoffice-writer libreoffice-gtk3.
