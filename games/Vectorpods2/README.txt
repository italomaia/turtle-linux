Vectorpods 2
===============

Team: None
Members: PyMike (with the help of Geometrian, Cthulhu32, X-Out, and mewo2)


DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/
  PyGame:     http://www.pygame.org/


RUNNING THE GAME:

Double-click on Vectorpods2.pyw


HOW TO PLAY THE GAME:

* Use the arrows to move and rotate.
* Press any other key to shoot.
* Press "a" to toggle antialiasing
* Press "p" to pause
* Press "Escape" to exit to the menu
* The object of the game is to shoot all the asteroids and ufos till you die.
