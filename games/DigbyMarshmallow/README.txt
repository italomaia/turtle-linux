Dependencies
============

You might need to install some of these before running the game:

  Python: http://www.python.org/ (version 2.5 or compatible)
  Pyglet: http://www.pyglet.org/ (version 1.1 or compatible)
  AVBin: http://code.google.com/p/avbin (compatible with pyglet)


Running the Game
================

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.
Otherwise open a terminal / console and "cd" to the game directory and run:
  python run_game.py


How to Play the Game
====================

From the menu, select start game and then click the diary icon to start the
first cutscene. Use the arrow keys to control, read the signs in game to learn
more.
